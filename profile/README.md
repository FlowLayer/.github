# FlowLayer

FlowLayer is a local-first orchestration tool for multi-service development environments.

It helps developers run, observe, and control complex local stacks with a predictable workflow:
- deterministic startup waves
- readiness checks between services
- real-time logs
- a WebSocket protocol for orchestration events and control
- an official terminal client
- JSONC configuration for readable, reviewable environment setup

FlowLayer is under active development.

Website: https://flowlayer.tech
