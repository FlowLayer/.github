# FlowLayer GitHub organization files

This repository contains GitHub organization-level files for FlowLayer.

## Organization profile

The public GitHub organization profile is defined in:

`profile/README.md`

It is displayed on the FlowLayer organization overview page:

https://github.com/FlowLayer

## Purpose

This repository is not a product repository.

It exists to host organization-level GitHub metadata, such as:

- the public organization profile
- shared issue templates
- shared pull request templates
- security policy files
- community health files
- other GitHub configuration files used across the FlowLayer organization

## Main project repositories

The main FlowLayer repositories are:

- `flowlayer`: public protocol, configuration, documentation, releases, and issue hub
- `tui`: official terminal client
- `site`: public website and documentation site
- `distribution`: installer and distribution channel material
